## 🚀 三步快速开始

### 第一步：复制背景图片

1. **打开文件浏览器**，进入 `d:\天府\玄荒道府\tianfu\`
2. **创建 `images` 文件夹**（如不存在）
3. **从 `d:\天府\玄荒道府\images\` 复制以下文件：**
   - `d1.png` → `d:\天府\玄荒道府\tianfu\images\d1.png`
   - `c1.png` → `d:\天府\玄荒道府\tianfu\images\c1.png`
   - `e1.png` → `d:\天府\玄荒道府\tianfu\images\e1.png`

### 第二步：安装依赖

1. **打开命令行**（PowerShell 或 CMD）
2. **进入项目目录：**
   ```powershell
   cd d:\天府\玄荒道府\tianfu
   ```
3. **安装依赖：**
   ```powershell
   npm install
   ```

### 第三步：启动开发服务器

```powershell
npm run dev
```

✅ 完成！浏览器会自动打开 `http://localhost:5173`

## ❓ 常见问题

### Q: 启动时提示 "npm 不是内部命令"
A: 确保 Node.js 已安装。检查：
```powershell
node --version
npm --version
```

### Q: 图片无法显示
A: 确保已复制图片到 `images/` 文件夹。详见 `IMAGES_SETUP.md`

### Q: 页面加载很慢
A: 这可能是第一次加载。Vite 会自动优化。稍后会很快。

### Q: 如何修改页面内容
A: 编辑 `src/views/` 下对应的 `.vue` 文件

### Q: 如何添加新的菜单项
A: 编辑 `src/components/AppSidebar.vue` 中的 `NAV_ITEMS` 数组

### Q: 如何更改颜色主题
A: 编辑 `src/assets/styles/global.css` 中的 `:root` CSS 变量
